This model aggregates customer order behavior.
Each row represents one customer and includes total orders, revenue, and average order value.

### Key Fields:
- **customer_id**: Primary Key — unique identifier for each customer.
- **number_of_orders**: Count of distinct orders.
- **total_revenue**: Total revenue generated by the customer.
- **avg_order_value**: Average order value (AOV).
