{% docs mrt_sales_localbike__sales_performance %}

This mart provides an overview of daily sales performance by store, staff, and product.

It includes:
- **order_date**: Date of the order.
- **store_id**, **staff_id**, **product_id**: Granularity dimensions.
- **total_orders**: Number of orders placed.
- **total_quantity_sold**: Total units sold.
- **total_revenue**: Total revenue generated.
- **avg_discount**: Average discount applied.

Use this mart to track sales trends, compare performance across stores, and monitor discounts impact.
